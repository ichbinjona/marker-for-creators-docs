# Marker Notes

Markers tell you **where** something happened. Notes tell you **what** happened. Together, they let you find any moment in hours of footage in seconds - without watching a single frame.

## The Note Bar

The Note Bar is the text field at the bottom of the recording screen. It's always visible, but works slightly differently depending on which page you're on.

### On the Marker Page (Main Recording Screen)

The Note Bar here is a simple text field. Tap it, type your note, and decide where to attach it.

**Two arrows appear when you type:**

- **Right arrow (→)** - Send to next marker. Your note is saved and will be automatically attached to the next marker you set. After tapping, the keyboard closes and a status message reminds you: "Set marker to add note." When you tap any marker button, the note is attached and the status clears.

- **Left arrow (←)** - Send to previous marker. Your note is attached to the last marker you placed. If that marker already has a note, you'll be asked to confirm before overwriting.

This is designed for two scenarios:

1. **You know what's about to happen** (e.g., the host says "next topic: budgeting") - type the note first, send right, then set your marker when the moment starts.

2. **Something just happened** (e.g., an unexpected guest reaction) - set your marker first to capture the timecode, then type the note and send left.

::: tip
The Note Bar supports multiple lines of text - just press Return to add a new line. Notes can be up to 3 lines in the Note Bar.
:::

### On the History Page

The Note Bar on the History Page works differently - and this is intentional.

**Tapping the Note Bar immediately creates a marker.** The moment you tap the text field, a new marker appears in the list at the current timecode. Now you can type your note and tap the checkmark (✓) to save it.

Why does it work this way? Because on the History Page, you're reviewing your markers. If you want to add a note, you probably want to mark this exact moment - so Marker creates the marker instantly, anchoring the timecode to the second you tapped, and gives you time to type.

**Choosing the marker color:**

- The **left button** shows a color circle after the marker is created - tap it to change the color
- **Long-press the left button** to choose which color the Note Bar uses by default:
  - A specific button (1, 2, 3, or 4) from your current preset
  - "Last Set Marker" - uses whatever marker you placed most recently
  - "Last NoteBar Marker" - uses the same color as the last marker created via the Note Bar

**Inline editing on marker cards:**

You can also tap "Add Note" directly on any marker card in the history list. This opens an inline text field right on the card - type your note and tap away to save. This hides the Note Bar while you're editing.

### On the Timeline Page

The Note Bar on the Timeline Page is not for notes - it's for creating timelines. Type a name and tap the checkmark to create a new timeline with that name. Or tap the plus (+) button to create a timeline with an auto-generated name.

On this page, pressing Return submits the timeline name (unlike the other pages where Return adds a new line).

## When to Use Notes

Notes are most powerful when they help your editor (or future you) find moments without scrubbing:

| Situation | Good Note |
|-----------|-----------|
| Guest says something quotable | "Guest: 'I was broke at 25, now I run three companies'" |
| Camera issue you want to skip | "Focus lost for 5 seconds" |
| Topic change in conversation | "Switch to: budget tips" |
| B-roll idea during filming | "Get insert shot of hands" |
| Technical note for color grade | "Lighting changed - clouds" |

Notes appear in your editing software's marker labels, so keep them short and descriptive enough to be useful at a glance.
